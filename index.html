What is HTML??

HTML (Hypertext Markup Language) is the set of markup symbols or codes inserted in a file intended for display on a World Wide Web browser page. The markup tells the Web browser how to display a Web page's words and images for the user. Each individual markup code is referred to as an element (but many people also refer to it as a tag). Some elements come in pairs that indicate when some display effect is to begin and when it is to end.

HTML is a formal Recommendation by the World Wide Web Consortium (W3C) and is generally adhered to by the major browsers, Microsoft's Internet Explorer and Netscape's Navigator, which also provide some additional non-standard codes.

A markup language is a set of markup tags. HTML tags describe HTML documents. Each HTML tag describes different document content.

· The DOCTYPE declaration defines the document type to be HTML

· The text between <html> and </html> describes an HTML document

· The text between <head> and </head> provides information about the document

· The text between <title> and </title> provides a title for the document

· The text between <body> and </body> describes the visible page content

· The text between <h1> and </h1> describes a heading

· HTML paragraphs are defined with the <p> tag:

<p>Digital Media.</p>

· The text between <p> and </p> describes paragraph.

HTML Line Breaks

The HTML <br> element defines a line break.

Use <br> if you want a line break (a new line) without starting a new paragraph:

Example

<p>MY<br>Name is<br>Niru with line breaks</p>

HTML Styling Background

This is how you add a background

<body style="background-color:lightgrey">

HTML Bold and Strong Formatting

The HTML <b> element defines bold text, without any extra importance.

Example

<p>This text is normal.</p>

<p><b>This text is bold</b>.</p>

HTML <i> Tag

The content of the <i> tag is usually displayed in italic.

<p> The premier league team that I support is Manchester City, because they are the <i>best</i>.</p>.

HTML Links - Hyperlinks

In HTML, links are defined with the <a> tag:

Example

<a href="http://www.Hackcess.com/HTML/"> you could learn more about HTML</a>

Tables

Tables are defined with the <table> tag. Tables are divided into table rows with the <tr> tag. Table rows are divided into table data with the <td> tag. A table row can also be divided into table headings with the <th> tag.

Example

<table><tr><td>

HTML Lists

Making a bulleted list in Html

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets (small black circles).

<ul style="list-style-type:disc">

<li>Pizza</li>

<li>Burger</li>

</ul>

Ordered HTML Lists

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

The list items will be marked with numbers.

<ol>

<li>Pizza</li>

<li>Burger</li>

</ol>

HTML Images Syntax

In HTML, images are defined with the <img> tag.

The <img> tag is empty, it contains attributes only, and does not have a closing tag. The src attribute defines the url (web address) of the image:

Example

<img src="url" alt="some_text">

CSS Tutorial

Home Next Topic

1.

CSS (which stands for Cascading Style Sheets) is a language used to describe the appearance and formatting of your HTML.

2.

You can make your HTML see the CSS information by putting a <link> tag between the <head> … </head> tags of your HTML page. The <link> needs 3 attributes:

1. A type attribute that equals to “text/css”

2. A rel attribute that should always be equal to “stylesheet”

3. A href attribute that should point to the web address of your CSS file.

An example of this in action:

<link type="text/css" rel="stylesheet" href="stylesheet.css"/>

3.

Inline/Internal styling with HTML:

e.g.

<p style="color:red">Hey There!</p>

This first is to put your CSS between <style></style> tags, right in the same file as your HTML.

4.

An example of the syntax for CSS looks like so:

selector {

property: value;

}

A selector can be any HTML element, such as <p>, <img>, or <table>. You just take off the <>s.

A property is the aspect of a selector. For example, you can change the colour, font-size, font style (font-family).

A value is a possible setting for a property, such as, color can be red, black, #FF0000 and so on.

Then you end each property value with a semi colon (;).

An example of all this in action is:

P {

font-size: 12px;

}

5.

It’s a good idea to write comments as you go along as it will help remind you why did something a certain way. In CSS, comments look like this:

/* An example of a comment */
